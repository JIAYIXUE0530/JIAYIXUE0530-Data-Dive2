# AI Stock Picker

## 触发条件

- 用户要求"股票分析"、"市场分析"、"选股"
- 用户使用 `/pick-stocks` 命令
- 用户表达"今天看好什么"、"市场观点"等意图
- 定时每日执行

## 执行流程

Phase 1 → Phase 2 → Phase 3 → Phase 4 → Phase 5
市场扫描   技术分析   基本面分析   生成推荐   数据存储

### Phase 1: 市场扫描
- 获取主要指数表现（S&P 500, NASDAQ, DOW）
- 扫描11个主要行业板块
- 识别市场情绪指标（VIX, 市场广度）
- 检测资金流向

### Phase 2: 技术分析
- 趋势分析：MA(20, 50, 200), MACD
- 动量指标：RSI, Stochastic
- 成交量分析
- 支撑/阻力位识别

### Phase 3: 基本面分析
- 行业估值水平（P/E, PEG, P/B）
- 盈利增长趋势
- 宏观经济因素
- 行业周期位置

### Phase 4: 生成推荐
- 看好行业/ETF：3-5个
- 看空行业/ETF：3-5个
- 每个推荐包含：
  - 置信度（1-10）
  - 3-5个核心理由
  - 风险因素
  - 关键价位

### Phase 5: 数据存储
- 追加到 recommendations.json
- 更新 performance.json
- 生成日报（可选导出CSV）

## 分析框架

### 多维度评估模型

**技术面（权重30%）**
- 趋势强度
- 动量信号
- 成交量确认
- 形态识别

**基本面（权重40%）**
- 估值吸引力
- 盈利增长质量
- 行业景气度
- 政策环境

**资金面（权重30%）**
- 机构持仓变化
- ETF资金流向
- 期权市场情绪
- 内部人交易

### 偏差检验（必须）

- **确认偏差**：是否只看支持观点的证据？
- **近因效应**：是否过度重视近期表现？
- **羊群效应**：是否盲目跟随市场共识？
- **锚定效应**：是否过度依赖某个价格水平？

## 数据存储

### recommendations.json
存储所有历史推荐，用于回测和准确率分析

### performance.json
跟踪预测准确率：
- 看多准确率
- 看空准确率
- 平均置信度
- 高置信度预测表现
