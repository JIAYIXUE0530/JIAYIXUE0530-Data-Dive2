# 抖音场日报分析风格参考

## 分析框架

### 一、分层分析结构
1. **价格带分层**：低价、中价、正价
2. **学段分层**：低幼、小学、初中、高中
3. **学科分层**：语文、数学、英语、编程、多科、其它

### 二、核心关注点

#### 1. 市场份额分析
- 谁吃了多少市场份额
- 市场集中度变化
- 头部竞品排名变化

#### 2. 销量变化分析
- 日销量数据（具体数字）
- 环比变化（百分比）
- 趋势判断（起量、回落、发力、冲量）

#### 3. 爆品分析
- 当前爆品是什么
- 爆品销量表现
- 爆品生命周期（冲量→回落）

#### 4. 竞品动态
- 新品上市情况
- 停播/恢复直播
- 投放策略变化

#### 5. 头部IP表现
- IP达人排名
- IP销量变化
- 新IP入局情况

### 三、洞察表达风格

#### 1. 量化数据为主
```
- 日销X单
- 环比+X%
- 市场份额X%
- 7日日均X单
```

#### 2. 对比分析
```
- GT超越YFD成为赛道top2
- ZYB（日销X单）> YFD（日销Y单）> GT（日销Z单）
- 吃了五大X%的市场份额
```

#### 3. 趋势判断关键词
```
- 起量：销量开始明显增长
- 回落：冲量后销量下降
- 发力：加大投放力度
- 冲量：短期内大幅增长
- 稳定：维持在一定水位
- 收缩：销量明显下降
```

#### 4. 产品细节描述
```
- 价格：X元
- 服务期：X天
- 课程内容：X节直播+Y节录播+Z本书
- IP号：XXX老师
```

### 四、典型句式模板

#### 市场份额句式
```
- [品牌]目前吃了[学段][价格带]五大X%的市场份额
- [品牌]超越[竞品]成为赛道top2
- [品牌]依旧是赛道头部（市场份额占X%）
```

#### 销量变化句式
```
- 由X日X单→昨日X单，日环比+X%
- 昨日冲量至X单后，已回落至X单
- 维持日销X单水位
```

#### 爆品分析句式
```
- 当前的爆品是X元[产品名]（昨日销量X单）
- [产品]开始起量，由X日X单→昨日X单
- [产品]持续发力，维持日销X单
```

#### 竞品动态句式
```
- [品牌]昨日推新品，入局[赛道]
- [品牌]恢复直播，昨日销量X单
- [品牌]假期停播X天，昨日恢复至日销X单水位
```

### 五、分析示例

#### 示例1：市场份额分析
```
小学：ZYB 目前吃了小学低价五大66%的市场份额，昨日日销8.3k单。
当前的爆品是39元小学数学画图法（昨日销量3.4k单）和49元小学大通关（昨日3.1k单）
```

#### 示例2：销量变化分析
```
初中：GT吃了初中低价38%的市场份额，当前爆品依旧是大胜满分物理（昨日1.3k单）
```

#### 示例3：竞品对比分析
```
高中：ZYB 依旧吃了高中五大60%的量，目前整体量级还在涨，日销3.7k单，日环比+32%
```

#### 示例4：新品分析
```
YFD 昨日推新品，入局抖音小学语文中价赛道。新品为299元"30天当堂写出满分作文"，
30天服务期，含8讲16课时真人直播课+6本实体书&16册练习册+240分钟配套视频，
但昨日仅日销9单
```

### 六、洞察生成要点

1. **数据驱动**：每个洞察必须有具体数据支撑
2. **对比鲜明**：通过对比突出变化和差异
3. **简洁有力**：一句话说清楚核心信息
4. **关注变化**：重点分析增量、减量、排名变化
5. **产品细节**：包含价格、服务期、课程内容等关键信息
6. **趋势判断**：用专业术语描述市场动态

### 七、优先级排序

1. **高优先级**：市场份额变化、头部竞品排名变化、爆品起量
2. **中优先级**：销量环比变化、新品上市、IP表现
3. **低优先级**：稳定维持的产品、无变化的产品
